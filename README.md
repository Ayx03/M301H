# M301H
移动魔百盒 M301H 刷机包等资源

把 update.zip 放在 U 盘根目录，关闭电源，把 U 盘插在网口旁的 USB 接口，用卡针戳进机身右侧的针孔并按住的同时打开电源，用遥控器选择第二项 `Apply Update from External Storage`，如果显示系统正在升级即表示成功，等待刷入完成后设备自动重启即可。
